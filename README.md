# real-time-face-mood-detection
📌 Overview

This project implements a real-time facial emotion recognition system using Convolutional Neural Networks (CNN) and OpenCV. It can detect emotions such as happy, sad, angry, fear, and surprise from live webcam feed or uploaded images.

---

🚀 Features

- 🎥 Real-time face detection using OpenCV
- 🧠 Emotion classification using CNN
- 📷 Supports both webcam and image input
- ⚡ Fast and efficient prediction

---

🛠️ Technologies Used

- Python
- OpenCV
- TensorFlow / Keras
- NumPy

---

📂 Project Structure

- "app.py" → Main application file
- "model.h5" → Trained model (Download from link below)
